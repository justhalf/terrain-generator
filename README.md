Terrain Generator
=================

Generates a nice random terrain consisting of land and water bodies.

Compile using:
``
mvn compile package
``

and run the produced jar:
``
java -jar target/terrain-generator-1.0.jar
``

Features
========
* Multiple generation methods
* Click-and-drag to move around
* Scroll to zoom
